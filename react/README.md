# Executar aplicação


## Instalação / execução
1. instalar react
```sh
  $ npm install -global create-react-app
```

2. criar projeto

```sh
  $ create-react-app huntweb
```

2. iniciar projeto

```sh
  $ yarn start
```

3. instalar biblioteca axios (utilizado para fazer a requisição com a API)
```sh
  $ yarn add axios
```
