# Node
Curso da Rocketseat.

# Executar aplicação
npm run dev

## Instalação
1. iniciar projeto
  npm init -y

2. instalr express
  npm install express

3. intalar nodemon em tempo de desenvolvimento - reiniciar o servido automaticamente.
  npm install -D nodemon  
  a) adicionar na tag script do package.json
     "dev": "nodemon server.js"
